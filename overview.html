Project Objectives

The overall objective of the PACE project is to deliver highly efficient software programmable accelerators based on Coarse-Grained Reconfigurable Architecture (CGRA) that can offer near ASIC-like compute efficiency while allowing full software programmability.

Concrete Goal

Bring server-class processing capability to resource-scarce edge devices at ultra-low power through a synergistic cross-layer approach. We expect to reach up to 50X improvement in compute efficiency compared to general-purpose processors in state-of-the-art edge devices.

PACE Architecture Overview

The PACE accelerator features an 8×8 matrix of processing elements (PEs) with 8 data memory modules, co-optimized for edge computing workloads. Each PE supports 17 instructions carefully selected through analysis of major kernels for edge AI applications.

PACE Accelerator Block Diagram and Supported Instructions

PACE-V SoC: RISC-V Integration

To maximize efficiency, PACE is integrated with a RISC-V processor (Rocket CPU) in a complete System-on-Chip design. This eliminates external microcontroller overhead, providing high-bandwidth data transfer with on-chip SRAM (512KB) and optimized boot flow.

PACE and RISC-V Core Integration Architecture

Chip Implementation

The PACE 1.0 chip was successfully fabricated in December 2023 using UMC 40nm CMOS technology. The chip measures 3.96mm × 2.4mm with PACE occupying about 30% of the die area, achieving 360 GOPS/W measured efficiency.

PACE 2.0: Next-Generation Design

PACE 2.0 represents a significant evolution, targeting 2000 GOPS/W efficiency through advanced SIMD architecture with 8-lane floating-point operations and integrated Address Generation Units. Designed in 12nm technology, it achieves 38% improvement over state-of-the-art.

PACE 2.0 Chip Layout (1.04mm² in 12nm, 8×8 PE Array)

Morpher: World-Class Compiler Framework

Morpher is our open-source integrated compilation and simulation framework for CGRA accelerators, offering quality mapping with 13× faster compilation compared to state-of-the-art approaches. Adopted by 10+ research groups worldwide, it supports end-to-end design flow from architecture specification to hardware validation.
