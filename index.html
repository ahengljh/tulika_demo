Research Overview

We are developing next-generation Coarse-Grained Reconfigurable Architectures (CGRAs) that bridge the gap between flexibility and efficiency in edge AI computing.

Our research focuses on creating an open ecosystem that includes hardware architectures, compiler frameworks, and design tools to accelerate innovation in domain-specific computing. Through projects like HyCUBE, PACE, and Morpher, we're building the foundation for agile hardware-software co-design.

Core Research Projects

HyCUBE

Architecture

A novel CGRA architecture featuring a reconfigurable single-cycle multi-hop interconnect that revolutionizes data movement efficiency. HyCUBE addresses the fundamental challenge of balancing flexibility with performance in reconfigurable computing.

Key Features:

  • Single-cycle multi-hop data routing
  • Configurable processing elements
  • Optimized for AI workloads
  • Scalable architecture design
2.5x Performance Gain
40% Energy Reduction

PACE

Silicon Implementation

A silicon-proven HyCUBE implementation achieving industry-leading power efficiency for edge AI applications. PACE demonstrates breakthrough performance in real-world deployments, including integration with WalkingWizard EEG systems for brain-computer interfaces.

PACE 1.0 Achievements:

  • 360 GOPS/W efficiency at 0.6V operation
  • 105 MHz maximum frequency
  • 43mW power consumption at typical load
  • WalkingWizard EEG integration demonstrated

PACE 2.0 Roadmap:

  • 2000 GOPS/W target with 12nm process
  • SIMD architecture enhancement
  • Advanced power management features
360 GOPS/W (v1.0)
2000 GOPS/W (v2.0 target)

Morpher

Open-Source Framework

An open-source, end-to-end CGRA toolchain that unifies design exploration, compilation, and validation. Morpher enables rapid prototyping and benchmarking of CGRA architectures, fostering innovation through community collaboration and standardized interfaces.

Framework Components:

  • Unified architecture description language
  • LLVM-based compilation framework
  • Cycle-accurate architectural simulator
  • RTL generation and verification tools
  • Comprehensive benchmark suite

Community Impact:

  • Active contributor community
  • Adopted by 10+ research groups
  • Extensible plugin architecture
100% Open Source
