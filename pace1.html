Technical Specifications

Process Technology UMC 40nm CMOS
Architecture 8x8 PE array (64 processing elements)
Die Size 4mm x 2.4mm (PACE takes ~30% of total SoC)
Peak Performance 142 GOPS/W @ 0.9V, 462 GOPS/W @ 0.5V, 568 GOPS/W @ 0.45V
Instructions per PE 17 optimized operations
Memory Configuration 8 data memory modules (8KB each)
Router Architecture Buffer-less single-cycle multi-hop

RISC-V SoC Integration

PACE 1.0 is integrated with a RISC-V processor (Rocket CPU) in a System-on-Chip configuration to maximize efficiency and minimize data transfer overhead.

Rocket CPU Features:

  • 5-stage in-order scalar core
  • RV32G/RV64G ISA support
  • Memory Management Unit (MMU)
  • Non-blocking data cache
  • Branch prediction (BTB, BHT, RAS)
  • Custom BootROM firmware (~200 bytes)
  • I2C EEPROM boot support

Power Optimizations

  • 19.06% power reduction through improved ALU and memory integration PE power consumption reduced to ~700 µW at 0.9V, 100 MHz
  • 24% idle power reduction with NOP instruction and clock-gating Individual tiles can be clock gated by the compiler

FPGA Emulation Platform

Development Platform

PACE RTL design successfully migrated to Digilent Genesys 2 FPGA (Xilinx Kintex-7™ XC7K325T-2FFG900C) for verification and application demonstration.

  • Clock generator mapped to MMCM module
  • SRAM mapped to block RAM
  • Two FPGA boards deployed (IME and NUS)
  • Successfully verified Array Add and MicroSpeech applications

Area and Power Breakdown

Area Distribution

  • PACE accelerator: 30% of SoC die
  • 64 PEs: 71% of PACE area
  • Configuration memory: 47% of PE area
  • Data memory: 29% of PACE area

Power Distribution

  • PEs: 97% of PACE power during processing
  • Configuration memory: 52% of PE power
  • Tile controller: 23% of PE power
  • Router: 14% of PE power
