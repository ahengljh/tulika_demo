Key Architectural Innovations

SIMD Architecture

Single-Input-Multiple-Data design for higher throughput and efficiency

  • 8-bit eight lane SIMD operations per PE
  • Data-width increased from 16bit to 64bit
  • Eight parallel 8-bit floating point operations

Advanced Memory System

Near-memory computing with intelligent data management

  • Near-memory compute macro
  • 80KB total memory capacity
  • Optimized data locality

Address Generation Unit

Hardware acceleration for memory access patterns

  • 75% improvement in initiation interval
  • Parallel Load/Store address calculation
  • Stride and constant address modes

Power Management

Fine-grained control for maximum efficiency

  • Dynamic voltage and frequency scaling
  • Per-PE power gating
  • Intelligent workload distribution
